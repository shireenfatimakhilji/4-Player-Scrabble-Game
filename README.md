# 4-Player Scrabble Game (Python)

A graphical, four-player Scrabble game developed in Python using Turtle Graphics.  
This project demonstrates modular programming, GUI design, game logic, and score tracking.

---

## Features
- Four-player gameplay with name input for each player
- Each player starts with **8 letter tiles** ("gotis")
- Players can select letters to place on the board
- If a player has a blank tile (`-`), they are prompted to choose their own letter
- Turn-based system: press **1** to finish a turn
- Score tracking for each player
- Game ends when tiles run out or all players have completed their turns
- Highlights winner and stores score history in JSON files
